# Google Colab 代码仓库

这个仓库用于存放在 Google Colab 中执行的代码和笔记本。

## 目录结构

```
colab/
├── notebooks/        # 存放你的工作笔记本
├── examples/         # 示例笔记本和教程
└── README.md         # 本文档
```

## 快速开始

### 1. 在 Colab 中打开笔记本

点击下面的徽章可以在 Google Colab 中打开示例笔记本：

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/deluxebear/colab/blob/main/examples/example_notebook.ipynb)

### 2. 从 GitHub 直接打开

你可以使用以下 URL 格式在 Colab 中打开任何笔记本：

```
https://colab.research.google.com/github/deluxebear/colab/blob/main/路径/到/笔记本.ipynb
```

### 3. 上传新笔记本

1. 在 Google Colab 中创建或编辑笔记本
2. 选择 `文件` > `在 GitHub 中保存副本`
3. 选择此仓库并选择合适的路径
4. 添加提交消息并保存

## 使用建议

- 在 `notebooks/` 目录中存放你的工作笔记本
- 使用有意义的文件名，便于日后查找
- 在每个笔记本的开头添加说明性文字
- 定期提交和推送你的更改

## 示例

查看 `examples/` 目录中的示例笔记本，了解如何：

- 进行基础 Python 编程
- 使用 NumPy 和 Pandas 处理数据
- 使用 Matplotlib 进行数据可视化
