# 笔记本目录

这个目录用于存放你的 Google Colab 笔记本。

## 使用说明

1. 将你的 `.ipynb` 文件放在这个目录中
2. 使用有意义的文件名，例如：`data_analysis_2023.ipynb`
3. 在文件顶部添加描述性的 markdown 单元格
4. 可以创建子目录来组织不同类型的笔记本

## 在 Colab 中打开笔记本

你可以直接在 GitHub 中打开笔记本，或使用以下链接格式在 Colab 中打开：

```
https://colab.research.google.com/github/deluxebear/colab/blob/main/notebooks/YOUR_NOTEBOOK.ipynb
```

将 `YOUR_NOTEBOOK.ipynb` 替换为你的笔记本文件名。
